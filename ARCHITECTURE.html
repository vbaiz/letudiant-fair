API Routes
app/api/ · Server-side only · Supabase service role
POST /api/scans
Record a QR stand/session scan. Called by the camera scanner page. Triggers lead score recomputation (pending).
LIVE
POST /api/leads/[id]/export
Reveals student PII to exhibitor. Writes exported_at + exported_by to lead row.
LIVE
POST /api/leads/compute
Recomputes lead score for student+school pair using lib/scoring/leadScore.ts.
LIVE
GET /api/leads/export-csv
Bulk CSV download of all leads for a school. Used by exhibitor leads page.
LIVE
POST /api/webhooks/eventmaker
Receives EventMaker fair registration. Runs identity reconciliation (email match + fuzzy merge).
LIVE
GET /api/auth/callback
Supabase OAuth callback handler. Exchanges code for session, sets cookies.
LIVE

Business Logic — lib/ & hooks/
Shared across all pages and API routes
lib/supabase/client.ts
Browser Supabase singleton. getSupabase(). Used by all client pages.
lib/supabase/server.ts
Server-side client (cookies). Used by API routes and proxy.ts.
lib/supabase/database.ts
All typed DB helpers: getUser, getEvents, getSchools, createScan, upsertMatch, getLeadsForSchool, getAdminStats…
lib/supabase/types.ts
Full TypeScript row types: UserRow, EventRow, SchoolRow, ScanRow, LeadRow, MatchRow, GroupRow…
lib/scoring/leadScore.ts
Deterministic 0–100 score. 50% declarative (field/level match) + 50% behavioural (scan/swipe/dwell). 3 tiers: Exploring / Comparing / Deciding.
lib/analytics/track.ts
Event tracker → analytics_events table. 16 event types: swipe, scan, bookmark, page_view, reel_play…
lib/gdpr/consent.ts
writeConsent, revokeConsent, cascadeDelete. Age gate (<16 → parental approval). RGPD compliant.
lib/identity/reconcile.ts
EventMaker webhook: exact email match + fuzzy (Levenshtein) name+DOB merge for identity deduplication.
hooks/useAuth.ts
React hook. Exposes: user, profile, role, loading. Used by every protected page for auth state.
